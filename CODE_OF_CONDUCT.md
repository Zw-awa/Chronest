# 行为准则

中文 | [English](CODE_OF_CONDUCT.EN.md)

## 我们的标准

Chronest 贡献者应保持讨论技术化、尊重他人、表达直接。骚扰、威胁、歧视、恶意挑衅和破坏性行为都不被接受。

## 鼓励的行为

- 先讨论具体行为和事实，再讨论观点分歧
- 可以强烈反对方案，但不要攻击个人
- 反馈应尽量具体、可执行
- 尊重项目范围、安全边界和维护者决策

## 不可接受的行为

- 人身攻击
- 骚扰或歧视
- 故意扰乱讨论或提交流程
- 未经许可公开他人隐私信息
- 在已有明确结论后持续进行恶意争辩

## 执行方式

维护者可以在必要时删除评论、关闭讨论、拒绝贡献，或将相关参与者移出项目协作范围，以保护项目和社区。

## 报告方式

如需报告行为问题，请通过 GitHub 私下联系维护者，或参考 [SECURITY.md](SECURITY.md) 中的私下联系路径。
